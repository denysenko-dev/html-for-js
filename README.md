# Frontend Development Course — Homework Repository

Репозиторій для домашніх завдань. Маючи базу у верстці та npm, проходив курс для актуалізації знань, структурування бази та освоєння автоматизації.

🔗 **Live demo:** [roaring-douhua-1943e6.netlify.app](https://roaring-douhua-1943e6.netlify.app/)

---

## 🚀 Структура проєкту

* **unit_01 – unit_07:** Основи HTML5/CSS3, позиціонування (Таблиці, Float, Position).
* **unit_08 – unit_13:** Сучасна верстка (Flexbox, Grid, Семантика), адаптивність, псевдокласи та анімація.
* **unit_14 – unit_15:** Автоматизація збірки розробки за допомогою **Gulp.js**.
* **unit_16 – unit_19:** Комплексний проєкт (резюме) на базі фреймворку **Bootstrap**, верстка блоків та повна адаптація під мобільні.
* **unit_20-21:** Деплой проєкту на хостинг ([live demo](https://roaring-douhua-1943e6.netlify.app/)) та динамічний вивід інформації за допомогою **JavaScript**.

---

## 🌱 Гілки репозиторію

* **`main`** — вітринна гілка: кожен урок зібраний в окрему папку `unit_NN/` (`unit_01` … `unit_20_21`).
  Саме її варто дивитись, щоб побачити весь курс одразу.
* **`lesson-1` … `lesson-20-21`** — по одній гілці на кожен урок, це "сирі" знімки роботи над конкретним
  завданням (без реорганізації в `unit_NN/`). Після завершення уроку його фінальний стан переноситься
  в `main` у відповідну папку `unit_NN/`.
