# Frontend Development Course — Homework Repository

A repository for course homework. Coming in with a base in markup and npm, I took this course to refresh my knowledge, structure that base, and pick up build automation.

🔗 **Live demo:** [roaring-douhua-1943e6.netlify.app](https://roaring-douhua-1943e6.netlify.app/)

---

## 🚀 Project structure

* **unit_01 – unit_07:** HTML5/CSS3 basics, positioning (Tables, Float, Position).
* **unit_08 – unit_13:** Modern layout (Flexbox, Grid, Semantics), responsiveness, pseudo-classes and animation.
* **unit_14 – unit_15:** Build automation with **Gulp.js**.
* **unit_16 – unit_19:** A comprehensive project (resume) built on the **Bootstrap** framework, block layout and full mobile adaptation.
* **unit_20-21:** Deploying the project to hosting ([live demo](https://roaring-douhua-1943e6.netlify.app/)) and dynamic data rendering with **JavaScript**.

---

## 🌱 Repository branches

* **`main`** — the showcase branch: each lesson is assembled into its own `unit_NN/` folder (`unit_01` … `unit_20_21`).
  This is the one to look at to see the whole course at a glance.
* **`lesson-1` … `lesson-20-21`** — one branch per lesson, these are "raw" snapshots of the work on a given
  assignment (without reorganizing into `unit_NN/`). Once a lesson is finished, its final state is carried
  over into `main` under the corresponding `unit_NN/` folder.
